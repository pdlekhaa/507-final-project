# 507-final-project
This project fine-tuned a BERT model on YouTube comments about Zohran Mamdani and applied model to 3 YouTube channels to track sentiment over time. 

Workflow:

# 1. Acquire ~200 comments about Mamdani -- at least 100 positive, and at least 100 negative
# 2. Manually classify comments
# 3. NLP + pre-processing
# 4. Fine-tune BERT model
# 5. Find many YouTube comments about Mamdani 
# 6. Apply model to these comments to get a large dataset
# 7. Perform analysis of public sentiment over time towards Mamdani.
